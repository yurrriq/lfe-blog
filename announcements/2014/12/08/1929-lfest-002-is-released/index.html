lfest 0.0.2 Is Released

lfest 0.0.2 includes the addition of text-only responses in addition to the already-supported HTML and JSON responses. This addition was made in order to more readily support LFE web app developers writing for services such as Google App Engine that expect apps to implementparticular resource endpoints (e.g., /_ah/health) which return plain-text responses.

More information:

  • lfest repo
  • lfest 0.0.2
  • An LFE-Erlang-Docker example web app which uses lfest for its routing and responses
